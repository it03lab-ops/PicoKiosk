# PicoKiosk

**Kiosk-лаунчер для VR-шлемов PICO 4 (и, возможно, Meta Quest)**  
Основан на проекте [PicoZen](https://github.com/barnabwhy/PicoZen) от [barnabwhy](https://github.com/barnabwhy) с серьёзными доработками для ограничения доступа пользователей.

> ⚠️ **Неофициальный проект** — не связан с компаниями PICO или Meta.
> 🎯 Создан для ситуаций, когда шлемом пользуются разные люди (музеи, выставки, обучение).
> 🔒 Блокирует доступ к системным настройкам и домашнему экрану, разрешая запуск только выбранных приложений.

## 🎯 Отличия от PicoZen

| Функция | PicoZen (оригинал) | PicoKiosk (этот форк) |
| :--- | :---: | :---: |
| Показывать все приложения | ✅ | ❌ (только белый список) |
| Управляемый администратором белый список | ❌ | ✅ |
| Блокировка доступа к системным настройкам | ❌ | ✅ |
| Блокировка панели уведомлений / быстрых настроек | ❌ | ✅ |
| Запрет выхода в системный лаунчер (кнопка HOME) | ❌ | ✅ |
| Загрузка APK-файлов | ✅ | ✅ (можно отключить) |

## ⚠️ Совместимость и предупреждения

*   **Основная цель:** PICO 4 (проверено на PICO 4). Должен работать на PICO Neo 3.
*   **Meta Quest (Quest 2/3/Pro):** *Не тестировалось.* Так как оригинальный PicoZen, по заявлениям, работал на Quest, этот форк может запуститься и там, но **это не гарантируется**. Методы блокировки системного интерфейса на PICO и Meta Quest отличаются. Используйте на свой страх и риск.
